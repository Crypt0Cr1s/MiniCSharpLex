 La tabla de simbolos esta guardada en una hashtable, cada posicion de esta tiene una estructura de: IDENTIFICADOR,TIPO,DESCRIPCION,VALOR, AMBITO, 
 para poder realizar una mantenimiento eso quiere decir actualizacion de valor, se realiza un update a la poscion en la hashtable que tenga el IDENTIFICADOR a modificar.
 Este update se, realiza haciendo una busqueda en la hashtable para determinar la posicion donde se encuentra el IDENTIFICADOR y luego se modifica el valor que el parametro VALOR tiene.
